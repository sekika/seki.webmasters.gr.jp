102 [link] 究極のバックギャモンツール Snowie3 itikawa URL 2000/12/28 15:48
Snowie 3は世界最強のバックギャモンソフトとして知られている。
このソフトの最大のセールスポイントはそこではなく、バックギャモンの学習、研究用ツールとしての有用性にある。

ポジションの評価
まずある特定の局面を入力しそれをスノーウィーの3-ply（3手読み）で判定させてみる。これだけでも相当に正確な評価ができるがさらに正確な評価を得るためにロールアウトという機能が用意されている。バックギャモンで言うところのロールアウトは言ってみれば、シュミレーションのことで実際にその局面からスノーウィーにゲームをさせてみることで局面の評価をさらに正確に得られる。

マッチの評価
GamesGrid,ＦＩＢＳ,NetGammon　等のインターネットバックギャモンサーバー上ではゲームの記録（棋譜）を残すことができる。この棋譜をSnowie3に読み込ませるとマッチ全体を解析させることができて、プレイのレベルを評価させることができる。スノーウィーの指摘するムーブエラー、キューブエラーを一つ一つ見て勉強すれば強くなるのは間違いない。現在GamesGridではレーティング１８００以上の日本人プレイヤーが1８名いるが、そのうち１６名がSnowie3を所有しているという事実がその学習、研究用ツールとしての有用性、効用を証明している。

日本バックギャモン協会ではSnowie3を販売している。
価格は4万円（送料、消費税込み）。ちょっと高いと思う人もいるかもしれないが、ほとんどのSnowie３ユーザーがその有用性を考えれば安い買い物という評価をしている。
ご注文は　itikawa@backgammon.gr.jp　まで。

A,B は Box に 1 点払う Box:+2 A:-1 B:-1

そしてゲームが続行します。


１．Box がそのまま勝った場合

簡単にするため、Box はシングル勝ちをしたとしますと
Captain は Box にキューブが２なので 2 点払う

Box:+2(計+4) Captain:-2(-2)

A:+2(+1) Captain:-2(-4)

結局 Box:+4, Captian:-4, A:+1, B:-1

これであってますか。


2. Captain がそのまま（リダブルせずに）勝った場合

同様にシングル勝ちとして
Box と A はそれぞれ Captain に対して 2 点払う

Captain:+4(+4) Box:-2(0) A:-2(-3)

最終的に Box:0, Captain:+4, A:-3, B:-1


3. Captain がリダブルする場合

Box と A それぞれがテイクかパスの判断をするわけですか。
Box が両方のキューブのテイクパスを判断するのですか。


4. ビーバー（、ラクーン）

"Give & Take" されたキューブ(上の例では A が Captain
に譲渡したキューブ)を Captain がビーバーすることは
できるのですか。あと多分現実的にはありえないでしょうが
そのビーバを降りる判断はもちろん A にあるわけですか。
ビーバーが可能なら A はラクーンできますか。


5. コンサルテーション

上の例で A のコンサルテーション（Box に対する口出し）
は許されるのですか。（Partner の Captain に対する
コンサルテーションが許されるルールの場合）

　結果として、この場合、　Box　＋４、Captain　−３、A　０、B　−１になります
　Boxは、Captainから２点、A,Bから１点ずつ勝ってるので４点
　Captainは、２レベルのキューブ２個負けたので−４、ただし、
　Aからのエキストラキューブを引くときにAから１点もらっている
　ので、３点負けになります

　のパスでBoxに１点、エキストラ引かせるときにCaptainに１点
　払っているので合計は０になります

　マイナス１よりは期待値が高いと判断し、パスした人は、マイナス
　１よりも低いと考えているので、１点あげてもう１つ引いてもらう
　というものです

シュエットトーナメントの場合は、明らかにノーダブル。
ビーバーしてもキューブが死ぬだけなので、ビーバーする
意味はなし（かけたいときにダブルをかけてキューブを
殺す権利を保有しておく方が効果的）。なので、ノーダブル、
テイクでしょう。

マネーのときは、微妙です。直感では、まだダブルをかけ
たくない。次に青が何を振るか。実戦では、厳密な計算を
する時間はないであろうことを考慮の上、アバウトに計算します。

１ぞろを振るときを含めて、１を振る確率と、１をふらずに
２を振る確率、１も２も振らない確率がそれぞれ3分の１
くらいあるとします。１を振ってから、2枚ともヒットできる
確率はかなり高いので、2枚ヒットできる確率、1枚ヒット
する確率、1枚もヒットできない確率をそれぞれ3分の１程度
としてみます。その後、1ポイントの白をヒットして、
白が生き返って、再び青がバーに乗る、といったような
やりとりを考えると、3枚クローズできる可能性は3分の１
よりもあがりそうですが、実際にはクローズアウトを完成
できずに逃げられることもありますので、3枚クローズ、
2枚クローズ、1枚クローズの確率を、これまた3分の１ずつ
程度に見積もっておきます。

3枚クローズの勝率が9割程度としますと、2枚クローズの
勝率は5割あるかどうかでしょう。1枚クローズだと、1割
あるかどうか？その平均で、勝率は5割程度に落ち着きそうな

　24ポイントをヒットして，1ポイントもついでにヒット。
　勝率は1-3で73%、1-6で74%。

(3)1-2、1-4、1-5 を振る場合 (6通り）
　24ポイントをヒット。もう一つの目は，7ポイントを
　動かす場合が多い。6ならば7/1とヒット。
　勝率は66%（1-4の場合）。

(4)2-2、2-3、2-6を振る場合 (6通り)

　できる。いずれも大体70%の勝率。

4-5 31%

3-4 43%
5-6 42%

となります。
