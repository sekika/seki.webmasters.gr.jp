バックギャモンソフト

PC ソフト

ここで紹介するソフトは、いずれも世界トップレベルのプレイヤーを凌駕する実力を持っています。初めは低いレベル設定から始め、ゲームに習熟してきたら段階的にレベルを上げていくことをお勧めします。これらのソフトは、単に「強い」だけでなく、機能性と操作性の両面で優れています。また、コンピュータとの対戦だけでなく、対局の分析や、プレイ中の不適切な判断に対する助言機能なども備えています。

ソフト名 プラットフォーム 補足説明
BGBlitz Windows, Mac, Linux 版がある。
最もおすすめ。BGBlitzの使い方を書いた。バージョン 3.2.0 以降は、思考エンジンに gnubg を選ぶことができるようになった。BGBlitz と gnubg で対戦させることもできる。日本語対応。
GNU Backgammon (gnubg) Windows, Linux, Mac
Mac へのインストールは、MacPortsを使う。 		インストールが問題なくできれば、とても快適。日本語対応。
eXtreme Gammon (XG) Windows 現在、トッププレイヤー達の多くはこのソフトで棋譜の解析をしているようであり、業界標準の地位を確立している。ただし、私は Mac ユーザーなので使っていない。英語のみ。

歴史的には、TD-Gammon が先駆者であり、その後 Jellyfish や Snowie がよく使用されていましたが、いずれも開発は終了しています。

モバイルアプリ

ここに紹介するアプリは、いずれも最強の思考エンジンを使い、プレイ中のヒント機能を備えている高機能なものです。個人的には、マウスよりもタッチで操作する方が楽なので、最近はPCソフトはあまり使わず、ほぼiPadで遊んでいます。

アプリ名 プラットフォーム 補足説明
XG Mobile Backgammon Android, iOS eXtreme Gammon のモバイル版。
True Backgammon Android, iOS, iOS (HD) アプリ内課金で BGBlitz の思考エンジンを入れることができる。iOS版ではGame Centerでオンライン対戦ができるが、あまり人はいない。
