詰将棋解答（作意手順）

たとえばＡコースの1番であれば、作品番号A01のリンクをクリックすると、 詰手順を再生することができます。解答は、Kifu for Java を使って再生します。

各解答ページに「棋譜保存」のリンクを用意してありますので、Javaを 実行できない環境であっても、棋譜ファイルをダウンロードして、 柿木義一のホームページページ の将棋棋譜管理シリーズ (KIFUS) で再生することが可能です。

掲載誌については、一通り過去の詰パラをチェックしてみましたが、まだ 見落としがありそうです。1998年6月以降の詰パラは手元にあるのですが、 それ以前のものが行方不明です。そこで、B04のように二回同じ作品を投稿する、 というへまをしてしまったわけですが、「未発表？」となっている作品の中に、 すでに発表されているけれど見落としているものがあるかもしれません。
